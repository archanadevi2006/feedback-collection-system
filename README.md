# Feedback Collection System

A Full Stack Feedback Collection System built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite

## Features

- User Feedback Form
- Admin Login
- Dashboard
- Search Feedback
- Filter by Rating
- Edit Feedback
- Delete Feedback

## Installation

```bash
cd backend
npm install
node server.js
```

Open the frontend using Live Server.
